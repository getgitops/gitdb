# Installation

## Requirements

- Node.js >= 20
- npm >= 10
- Git >= 2.20 (installed on system)

## Install Package

```bash
npm install @kettu/gitdb
```

## Peer Dependencies

GitDB no tiene dependencias externas. Solo requiere:
- Node.js runtime
- Git instalado en el sistema

## Verify Installation

```bash
npm list @kettu/gitdb
```

Deberías ver la versión instalada.
